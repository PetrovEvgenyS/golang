## Погода (CLI-приложение)

Небольшое консольное приложение на Go для получения текущей погоды по городу или по IP-адресу пользователя.

Используются внешние сервисы:
- `ipapi.co` — определение города по IP;
- `countriesnow.space` — проверка существования города;
- `wttr.in` — получение строки с погодой.

### Требования

- Go, совместимый с модулем `demo/weather` (см. `go.mod`).

### Установка и запуск

В каталоге проекта `Погода`:

```bash
go run .
```

По умолчанию приложение:
- определяет город по IP (`ipapi.co`);
- запрашивает погоду в текстовом формате `1` у `wttr.in`.

### Параметры командной строки

- `-city` — город пользователя. Если указан, сначала проверяется существование города во внешнем сервисе.
- `-format` — формат строки ответа сервиса `wttr.in` (целое число от 1 до 4).

Примеры:

```bash
# Погода по IP
go run .

# Погода для конкретного города
go run . -city="Moscow"

# Погода для города с другим форматом вывода
go run . -city="Moscow" -format=3
```

### Обработка ошибок

При ошибках определения местоположения или получения погоды приложение выводит человекочитаемое сообщение и завершает выполнение с ненулевым кодом возврата.
